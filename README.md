# 비이브코딩 포트폴리오 프로젝트

## 프로젝트 구조
```
portfolio/
├── index.html      # 메인 페이지
├── styles.css      # 스타일시트
└── script.js       # 자바스크립트 파일
netlify.toml        # Netlify 설정 파일
```

## 기술 스택
- HTML5
- CSS3 (반응형 디자인)
- Vanilla JavaScript

## 학습한 내용
- HTML5와 CSS3를 사용한 반응형 웹 디자인
- Vanilla JavaScript를 사용한 인터랙티브 기능
- Netlify를 사용한 정적 웹사이트 호스팅
- [배포 링크](https://reliable-dolphin-2147cd.netlify.app/)
- Git을 사용한 버전 관리
- GitHub을 사용한 코드 관리
